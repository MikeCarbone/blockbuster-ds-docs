## Link Wrapping
To use in Next.js or another toolkit that requires a custom parent for links, the button takes a `parentEl` prop that takes a JS component.
```js
import { Link } from 'next/link'

export default function Home => (
    <Button parentEl={ Link }>
        Click
    </Button>
);
```

## Props

### alignSelf

Type: String <br />
Options: Any valid CSS property <br />
Default: center  <br />
Description: align-self CSS property  <br />

### margin

Type: String <br />
Options: Any valid CSS property <br />
Default: 0  <br />
Description: margin CSS property  <br />

### secondary

Type: Boolean <br />
Options: true | false <br />
Default: false  <br />
Description: Whether to render the primary or secondary button  <br />
