## Adding Links
Links can be added by passing them in as children. So for example:
```js
import { Header, Link } from 'blockbuster-design-system'

<Header>
      <Link color="var(--white)" href="#">Home</Link>
      <Link color="var(--white)" href="#">About</Link>
</Header>

```

## Optional Button
The button will only render if you pass in the `buttonText` prop. Example without the button:
<Header>
      <Link color="var(--white)" href="#">Home</Link>
      <Link color="var(--white)" href="#">About</Link>
</Header>


## Link Wrapping
To wrap links in a Next.js project or otherwise, you can pass custom elements to wrap the links.
```js
import { Link } from 'next/link'

export default function Home => (
    <Header
      buttonText="Log In"
      buttonLink="/page"
      linkWrapper={Copy}
    />
);
```

## Props

### buttonText
Default: ""
Type: String

### children
Default: ""
Type: String

### buttonLink
Default: ""
Type: String
Description: Href for the button link

### linkWrapper
Default: {}
Type: React Component
Description: Component for link Wrapping
